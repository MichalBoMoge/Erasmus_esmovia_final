## Getting Started

Follow the steps.

### Prerequisites

To run this project you must meet the following requirements.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/michalbomoge/Erasmus_esmovia_final.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Install npm i
   ```js
   npm i
   ```
4. Run the project
   ```js
   npm run dev
   ```
